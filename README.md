# Android Calculator App

A clean and interactive Android Calculator application built with Java and XML. This project demonstrates basic mobile development concepts, proper UI layouts, and standard event handling in Android, designed with European IT industry standards in mind.

## 🚀 Features
* **Splash Screen:** A welcoming 2-second splash screen on launch.
* **Basic Arithmetic:** Supports Addition, Subtraction, Multiplication, and Division.
* **Decimal Support:** Handles floating-point numbers accurately.
* **Dynamic Display:** Separate sections for expressions and results.

## 🛠️ Technologies Used
* **Language:** Java
* **UI Design:** XML (Android Layouts)
* **Platform:** Android SDK
* **Version Control:** Git & GitHub

## 📦 Project Structure
* `MainActivity.java` - Contains the core application logic and button click events.
* `layout.xml` - The main user interface layout.
* `splash_screen.xml` - The splash screen layout shown on startup.
